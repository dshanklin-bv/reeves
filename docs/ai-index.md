## Why Raw Markdown?

1. **AI-Readable** - AIs can fetch and parse markdown directly
2. **Copy-Paste Ready** - Easy to paste into Claude Code or other AI tools
3. **Version Control** - Git-trackable, reviewable, diff-friendly
4. **No Rendering** - No need for HTML parsing or JavaScript execution
5. **Lightweight** - Fast to fetch and process
